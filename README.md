# HAJU SOFT - Official Corporate Homepage

This repository contains the official static homepage for **HAJU SOFT**, designed for direct deployment to **GitHub Pages**.

## 🚀 Quick Start / Deployment

This site is built as a zero-build static application. To deploy it to GitHub Pages:

1.  **Create a Repository**: Create a new public repository on GitHub named `hajusoft.github.io` (for user site) or any name for a project site.
2.  **Upload Files**: Push the contents of this folder to the `main` branch.
3.  **Enable GitHub Pages**:
    -   Go to **Settings** > **Pages** in your GitHub repository.
    -   Under **Build and deployment** > **Source**, select **Deploy from a branch**.
    -   Select `main` branch and folder `/ (root)`.
    -   Click **Save**.
4.  **Visit Your Site**: Your site will be live at `https://<your-username>.github.io/<repo-name>/`.

## 📂 File Structure

-   `index.html`: The core homepage file (HTML/Tailwind CSS/JS).
-   `hajusoft.png`: Official company logo.
-   `.nojekyll`: Disables Jekyll processing for faster and more reliable deployment.
-   `README.md`: Documentation and deployment guide.

## 🛠 Customization Guide

-   **Logo**: If you change the logo filename, update the `<img src="hajusoft.png">` tags in `index.html`.
-   **Contact Email**: Search for `hajusoft@outlook.com` in `index.html` and replace it with your official email.
-   **Services & Projects**: Edit the text content in the respective sections to reflect current offerings.
-   **Custom Domain**: 
    -   If using a custom domain (e.g., `hajusoft.com`), add a `CNAME` file to this directory containing only the domain name.
    -   Configure your DNS settings (A/AAAA/CNAME records) according to GitHub's documentation.
